# upskillcampus

# 🐍 Python Mini Projects Collection

A collection of beginner-friendly Python projects developed to strengthen programming fundamentals and practical problem-solving skills. These projects demonstrate concepts such as file handling, automation, user interaction, data management, and basic algorithm implementation.

---

# 📌 Projects Included

## 1. 📂 File Organizer

### Description
The File Organizer automatically scans a directory, identifies files based on their extensions, creates category folders, and moves files into their respective folders.

### Features
- Organizes files automatically
- Categorizes Images, Documents, Audio, Video, Archives, Code, and Others
- Creates folders automatically
- Displays file name before moving
- Uses Python file handling modules

### Technologies
- Python
- os
- pathlib
- shutil

---

## 2. 🔐 Password Manager

### Description
The Password Manager is a command-line application that allows users to securely save and view website login credentials using Python file handling.

### Features
- Add new passwords
- View saved passwords
- Stores credentials in a text file
- Simple menu-driven interface
- Handles missing files gracefully

### Technologies
- Python
- File Handling
- os Module

---

## 3. 🎮 Quiz Game

### Description
The Quiz Game is an interactive Python application that asks multiple-choice questions, checks user answers, and calculates the final score.

### Features
- Multiple-choice questions
- Score calculation
- Instant feedback
- Menu-driven interface
- Easy to add new questions

### Technologies
- Python
- Lists
- Dictionaries
- Loops
- Conditional Statements

---

## 4. 🔗 URL Shortener

### Description
The URL Shortener converts long URLs into short, easy-to-share links by generating unique identifiers and maintaining a mapping between original and shortened URLs.

### Features
- Shortens long URLs
- Generates unique short codes
- Retrieves original URLs
- Simple and efficient implementation
- Easy to extend with database support

### Technologies
- Python
- Random/String Modules
- Dictionary or Database Storage
